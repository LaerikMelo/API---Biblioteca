O programa utiliza as seguintes linguagens:
Python
Javascript
Html
CSS

utiliza a seguinte framework: Flask (python)

para instalar o flask, use o seguinte comando no CMD:

$ pip install Flask

para obter o token, execute o arquivo "gettoken.py", no qual será gerado um token no terminal
no arquivo "gettoken.py" há uma seção chamada 'Authorization', no qual o token deverá ser inserido


no Vs code, execute o programa no arquivo "app.py", que lhe dará um IP 

no meu caso: http://127.0.0.1:5000/ (abrir no navegador)
